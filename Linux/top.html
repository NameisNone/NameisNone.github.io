    # TOP性能监控工具

    命令参数：

    • -d：指定 top 命令的更新间隔时间（单位为秒）。
    • -n：指定 top 命令的刷新次数。
    • -p：指定要监控的进程 ID。
    • -u：指定要监控的用户。
    • M：按照内存使用量排序显示进程信息。
    • P：按照 CPU 使用率排序显示进程信息。
    • T：按照进程运行时间排序显示进程信息。
    • -H/H：查看线程。
    • 数字 1：查看详细cpu占用(多核显示)。
    • 字母 b/B：高亮/背光切换，配合x/y进行行或列选择：
      • y，行高亮；
      • x，列高亮；
      • shift+> 、shift+< 左右移动需要高亮的列，并进行排序。
